To-Do List App

Built a simple To-Do List application using React. Users can input tasks through an input field and add them to the list by clicking the "OK" button. The list displays tasks with checkboxes, allowing users to mark tasks as completed or delete them. The application uses local storage to persist tasks, ensuring that the data remains accessible even after refreshing the page. The project incorporates React's state and useEffect hook to manage and display tasks dynamically.
